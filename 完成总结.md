## 🔍 OCR 图片识别(可选)

如果需要识别图片中的文字,有以下选择:

### 方案 1: QQ/微信(最简单)
1. 打开网页,截图图片
2. 使用 QQ/微信的截图 OCR 功能
3. 复制文字

### 方案 2: 在线 OCR 服务
- 百度 OCR: https://cloud.baidu.com/product/ocr
- 腾讯云 OCR: https://cloud.tencent.com/product/ocr

### 方案 3: 本地 OCR(需要技术)
```bash
# 安装 PaddleOCR
pip install paddleocr paddlepaddle

# 使用提供的脚本
node scripts/run-ocr-on-notes.js
```
