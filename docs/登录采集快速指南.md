# 🚀 登录采集快速指南

## 三步开始使用

### 第一步:登录

```bash
npm run login
```

浏览器会自动打开,在浏览器中:
1. 使用手机号或扫码登录小红书
2. 登录成功后,脚本会自动保存 Cookie
3. 浏览器自动关闭

### 第二步:采集

```bash
npm run extract:login
```

脚本会:
- 自动加载保存的登录状态
- 访问每篇笔记并提取真实内容
- 自动过滤"用户协议"等虚假内容
- 保存到 `data/notes-with-login.json`

### 第三步:查看

```bash
npm run serve
```

在浏览器打开: http://localhost:8080

---

## 常用命令

```bash
# 登录保存会话
npm run login

# 检查 Cookie 状态
npm run login:check

# 使用登录会话采集
npm run extract:login

# 查看 Web 界面
npm run serve

# 生成文档
npm run doc:full
```

---

## Cookie 说明

- **保存位置**: `data/cookies.json`
- **有效期**: 约 30 天
- **安全性**: 已在 `.gitignore` 中,不会提交到 Git

---

## 需要重新登录?

如果采集时出现"用户协议",说明 Cookie 失效:

```bash
npm run login  # 重新登录即可
```
