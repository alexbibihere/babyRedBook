## 🔍 关于 OCR 图片识别(可选)

如果你想要识别图片中的文字,可以:

### 使用在线 OCR 服务

推荐使用:
- **百度 OCR API**: https://cloud.baidu.com/product/ocr
- **腾讯云 OCR**: https://cloud.tencent.com/product/ocr
- **QQ/微信截图**: 直接截图图片,使用内置 OCR 功能

### 或者使用 Python 脚本

如果需要批量识别,可以:
1. 使用 PaddleOCR: `pip install paddleocr`
2. 使用 Tesseract OCR
3. 调用在线 API
